# Ray Casting Engine

## Features

### Render 2d Map
- Read map from file
- Render player, and walls,
- Extras: support mutliple rooms

### Dual 2d 3d View
View both the 2d map and 3d map in the same screen in real time.

### Cast Rays
- support ray casting.
- handle constants gracefully.
- No fish eye.

### Textures
- Support texture loading.
- Map walls.
- Enemies

### Lighting
Create a lighting system.
